# Trabalho de Redes de Computadores - Sockets TCP

Este é um projeto simples de comunicação entre cliente e servidor usando **sockets TCP em Python**.

## 📁 Arquivos

- `servidor.py`: roda o servidor que espera conexões e responde mensagens.
- `cliente.py`: conecta ao servidor e permite enviar mensagens.

## 🚀 Como executar

1. Certifique-se que o **Python** está instalado no computador.
2. Baixe ou clone este repositório.
3. No terminal, abra a pasta onde estão os arquivos.
4. Em dois terminais diferentes:

### Terminal 1 (servidor):
```bash
python servidor.py
```

### Terminal 2 (cliente):
```bash
python cliente.py
```

5. Digite mensagens no cliente e veja as respostas do servidor.
6. Para encerrar, digite `sair`.

## 🎯 Objetivo

Demonstrar de forma simples o funcionamento da comunicação entre processos em rede usando o protocolo TCP com sockets.
